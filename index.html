installation

1 — add repository

replace x86_64 with your architecture. run xbps-uhelper arch if unsure. 

echo 'repository=https://github.com/zveihander/void-supplements/releases/download/x86_64' \
  | sudo tee /etc/xbps.d/90-void-supplements.conf
2 — sync & accept signing key

xbps will show the key fingerprint on first sync. verify it matches de:a4:97:6c:18:26:10:84:46:53:27:9a:20:5a:b3:bc before continuing. 

sudo xbps-install -S
3 — install
sudo xbps-install -S <pkgname>
optional — import key manually
curl -L https://zveihander.github.io/void-supplements/void-supplements.pub.pem \
  | sudo xbps-rindex --import-key /dev/stdin
