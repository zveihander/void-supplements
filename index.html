installation

1 — add repository

replace x86_64 with your arch. run xbps-uhelper arch if unsure.

echo 'repository=https://github.com/zveihander/void-supplements/releases/download/x86_64' \
  | sudo tee /etc/xbps.d/90-void-supplements.conf
2 — sync & accept signing key

xbps will show the key fingerprint on first sync. verify it matches void-supplements.pub.pem before accepting.

sudo xbps-install -S
3 — install
sudo xbps-install -S <pkgname>
optional — import key manually
curl -L https://zveihander.github.io/void-supplements/void-supplements.pub.pem \
  | sudo xbps-rindex --import-key /dev/stdin

notes

packages are built automatically on every push to master that touches packages/. each architecture is built in parallel inside a void linux container using xbps-src. arm targets are cross-compiled from x86_64. not every package supports every architecture — check the template's archs= field. all packages are rsa-signed.
