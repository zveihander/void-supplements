Quick Install

1

Add the repository for your architecture (example: x86_64 glibc)

echo 'repository=https://zveihander.github.io/void-supplements/packages/x86_64' \
           | sudo tee /etc/xbps.d/99-void-supplements.conf
2

Sync and accept the repository fingerprint

sudo xbps-install -S

When prompted, verify the fingerprint matches the one in this repo's void-supplements.pub.pem and press y.

3

Install packages as usual

sudo xbps-install -S <package-name>

Public Signing Key

All packages are signed with an RSA key. XBPS will ask you to accept it on first sync. You can also import it manually: 

curl -L https://zveihander.github.io/void-supplements/void-supplements.pub.pem | sudo xbps-rindex --import-key /dev/stdin
